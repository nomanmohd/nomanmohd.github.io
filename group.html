Data Security & Privacy Lab

In recent years, there has been a tremendous growth in personal data collection, such as health data, location data, financial data, search queries, social network data, and sensor data collected through smartphones and IoT devices. Data collection has raised various security and privacy concerns regarding storage, analysis, and sharing of personal information. In the Data Security & Privacy Lab, we design, implement and evaluate data management systems using techniques from cryptography, machine learning, and computer systems to improve data protection mechanisms for real-life application scenarios.
